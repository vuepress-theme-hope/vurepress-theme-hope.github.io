自定义对齐

  • 演示
  • 其他自定义容器

通过对 vuepress-plugin-container 进行额外的配置注入，你可以使用

::: center
要居中的段落
:::

::: right
要居右的段落
:::

来对你的段落对齐进行自定义。

# 配置







 





module.exports = {
  plugins: [
    [
      "md-enhance",
      {
        // 启用自定义对齐
        align: true,
      },
    ],
  ],
};
