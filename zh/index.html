vuepress-theme-hope

一个具有强大功能的 vuepress 主题✨

快速上手 💡 配置 🛠

Markdown 增强 🧰

新增文字对齐、上下角标、脚注、标记、数学公式、流程图与幻灯片支持

浏览量与评论 💬

配合 Valine 与 Vssue 来开启阅读量统计与评论支持

文章信息展示 ℹ

为你的文章添加作者、写作日期、预计阅读时间、字数统计等信息

博客支持 📝

为你的文章添加日期、标签和分类，即可自动生成文章、分类、标签与时间轴列表

文章加密 🔐

你可以为你的特定页面或特定目录进行加密，以便陌生人不能随意访问它们

主题色切换 🎨

支持自定义主题色并允许用户在预设的主题颜色之间切换

深色模式 🌙

可以自由切换浅色模式与深色模式

SEO 增强 ⚒

将最终生成的网页针对搜索引擎进行优化。

Sitemap 🗺

自动为你的网站生成 Sitemap

Feed 支持 📡

你可以生成你的 Feed，并通知你的用户订阅它。

PWA 支持 📲

内置的 PWA 插件会让你的网站更像一个 APP。

TS 支持 🔧

为你的 VuePress 开启 TypeScript 支持

更多新特性 ✨

包括博客主页、图标支持、路径导航、页脚支持、全屏按钮、返回顶部按钮等

提示

引入 config 函数的作用是在你编辑配置的时候，通过 TS 的 Interface 以及 JSDoc，很方便的给你提示。

同时，config 函数也会为你补全一些直接交付给 VuePress 的默认配置。

你可以查看 本文档配置 (opens new window) 作为一个配置参考。
